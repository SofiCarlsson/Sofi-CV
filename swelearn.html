SweLearn

Beskrivning

Detta projekt gjordes i samband med kursen CS50P - Introduction to Programming with Python via Harvards Onlineuniversitet via EDX.

SweLearn är en inlärningsapplikation där användare kan öva svenska på ett strukturerat sätt. Användaren kan välja att få instruktionerna på spanska eller på engelska och börjar därefter genom att välja en svårighetsnivå mellan 1 och 4. Nivå 1 handlar om substantiv som är djur. Nivå 2 handlar om verb i presens, nivå 3 handlar om substantiv i bestämd form och nivå 4 handlar om substantiv i bestämd form tillsammans med ett verb i presens. Efter att ha slutfört en nivå kan användaren välja att gå vidare till nästa nivå eller avsluta programmet.

All data lagras i en JSON-fil, och en separat testfil har skapats för att verifiera programmets funktionalitet.

Teknologier som används

  • Python
  • Pytest
  • JSON-lagring
  • emoji-library

Bilder från projektet

Engelska versionen

Instruktioner finns tillgängliga både på Engelska och på Spanska, som visas till höger.

Spanska versionen

Genom att använda ternary operators kunde jag på ett enkelt sätt kolla om användaren angivit engelska eller spanska som inmatningspråk och sedan välja rätt instruktioner.

Datalagring

Filen words.json innehåller all data som används i testet. Jag valde att använda en JSON-fil med dictionaries eftersom det gjorde det enklare att plocka ut relevant data eftersom den då var strukturerad enligt language (top level key), level samt keyvalue-pair word och word_swedish.

Pytest

Pytest användes för att säkerställa att all funktionallitet fungerade korrekt genom att testa funktioner med fördefinierade parametrar och förväntade resultat.

Kod för att välja språk

Funktionen get_language tar användarens indata som en parameter och validerar True eller False om indata är antingen eng eller esp. Funktionen öppnar json-filen och kontrollerar om user_input är ett av nyckelvärdena på toppnivå, vilket är antingen "eng" eller "esp". Om så är fallet returnerar funktionen True och indata är nu validerad. Om språket inte är giltigt returnerar det False till main och funktionen anropas om och om igen tills användaren matar in "eng" eller "esp".

Kod för att hämta ner giltigt språk, nivå samt hämta orden från json-filen

Funktionen get_level liknar get_language och tar användarens indata och validerar om det är en giltig nivå, dvs. 1, 2, 3 eller 4. Om leveln inte är giltig returnerar den False till main och funktionen anropas om och om igen tills användaren matar in rätt.

Funktionen get_words fungerar annorlunda eftersom den tar in de validerade variablerna språk och nivå. Denna funktion öppnar json-filen och tar ut korrekt data och returnerar ordboken med nyckel- och nyckelvärdeparen för den nivån.

Gallactic Getaway

Beskrivning

Gallactic Getaway är ett fiktivt resebolag som skapades i samband med kursen Klientprogrammering för webbsystem. Projektet utvecklades i grupp. Hemsidan är utformad för att efterlikna en verklig företagshemsida och låter användaren läsa om företaget, dess anställda, de olika resemålen samt kontakta företaget via ett kontaktformulär.

Teknologier som används

  • HTML och CSS
  • JavaScript

Bilder från projektet

Startsidan

Här är startsidan för hemsidan. Menyn har fyra flikar där kunderna kan läsa mer om företaget, dess anställda, erbjudna resemål samt kontakta företaget via ett kontaktformulär. Hemsidan är responsiv och fungerar därför både på datorer och mobilskärmar.

Om de anställda

Här kan kunderna läsa mer specifikt om varje anställd på företaget och deras roller. Det är dessutom möjligt att klicka sig vidare till de anställdas egna sidor för att läsa mer om varje individs erfarenheter och styrkor.

Destinationer

Här presenteras de olika fiktiva resemålen genom att använda containers och items för att sortera bilderna och bildtexterna på ett snyggt och tydligt sätt.

Bokningsformulär

Här presenteras bokningsformuläret där användaren kan boka en resa till ett av de valda resemålen eller ställa en fråga. Själva verifieringen görs genom att användarens inmatning kontrolleras med JavaScript, vilket visas nedan.

Valideringskod

Förnamn och efternamn valideras enligt standard för namn. E-post och telefonnummer valideras på liknande sätt, och inga fält tillåter tom inmatning.

Valideringskod

Vi valde att använda realtidsvalidering för att enklare hjälpa användaren att förstå om alla fält var korrekt ifyllda innan de behövde trycka på skicka-knappen. Valideringen kontrollerar att alla fält för förnamn, efternamn, e-post och telefon är korrekt ifyllda. Om något fält inte uppfyller kraven markeras det med en röd ring runt inputfältet.
